🚀 QUICK START:
===============

1. AUTOMATIC SETUP:
   Linux/Mac: ./setup.sh
   Windows: setup.bat

2. MANUAL SETUP:
   - Install Python 3.8+ and Node.js 16+
   - pip install -r requirements.txt
   - cd frontend && npm install
   - python seed_data.py

3. RUN APPLICATION:
   Backend: python main.py (http://localhost:8000)
   Frontend: cd frontend && npm start (http://localhost:3000)

4. LOGIN:
   Username: admin, Password: admin123
   Username: manager, Password: manager123

🆘 SUPPORT:
===========

If you encounter any issues:
1. Check README.md for detailed instructions
2. Review API docs at http://localhost:8000/docs
3. Check browser console for frontend errors
4. Verify all dependencies are installed
5. Use demo credentials provided

Common solutions:
- Delete node_modules and reinstall (frontend issues)
- Delete construction_ai.db and re-run seed_data.py (database issues)
- Check port availability (8000 for backend, 3000 for frontend)
- Ensure virtual environment is activated (Python issues)
